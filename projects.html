Excel Data Visualization Business Analysis

Dashboard Pergerakan Barang di PT Gallant Maju Jaya

Pengolahan data menggunakan Microsoft Excel dan visualisasi interaktif dengan Looker Studio untuk menyajikan insight bisnis dan tren penjualan secara lebih mudah dipahami.

🎯

Tujuan Project

Membuat dashboard monitoring gudang untuk memvisualisasikan aktivitas barang masuk, barang keluar, return, dan kondisi stok. Tujuannya adalah membantu proses pemantauan persediaan serta mendukung pengambilan keputusan terkait pengelolaan stok barang.

🗂️

Dataset

Dataset yang digunakan merupakan data simulasi (dummy data) yang dibuat dengan bantuan Claude AI. Dataset berisi 1.000 transaksi gudang dalam format CSV yang mencakup data persediaan, pergerakan barang, dan return.

🛠️

Tools yang Digunakan

Microsoft Excel untuk pengolahan dan perapihan data, Pivot Table untuk membuat ringkasan data, serta Looker Studio untuk membangun visualisasi data yang interaktif.

Key Insights

Temuan penting dari analisis data ini:

1

Arus Barang Masuk Lebih Tinggi dari Barang Keluar Persediaan masih cukup aman namun perlu dilakukan pemantauan agar tidak terjadi penumpukan stok pada slow moving product.

2

Transaksi Return Sangat Rendah, total return sejumlah 2.563 unit dari total 51.361 pergerakan barang.

3

Produk Terlaris Didominasi Kategori Kebutuhan Rumah Tangga

4

Terdapat Beberapa Produk yang Membutuhkan Restock
